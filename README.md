# PhotoSculpt-Textures
Here is the Open source version of my software PhotoSculpt Textures v2.4

# What is it?
* PhotoSculpt Textures is a specialized software tool used to create high-resolution 3D textures and models from just two photographs of a subject taken from different angles. 
* This software caters specifically to computer graphics artists by enabling them to generate fully textured 3D models and tileable textures for further use in various 3D applications.

# History
* 2010: Release of PhotoSculpt Textures v1.00
* 2019: End of commercial availability of the software.
* 2025: Release of the Open Source latest version of PhotoSculpt Textures v2.04
